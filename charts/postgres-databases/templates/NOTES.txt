To verify the deployments:

  kubectl get statefulsets -n {{ .Release.Namespace }}
  kubectl get pods -n {{ .Release.Namespace }}
  kubectl get services -n {{ .Release.Namespace }}
  kubectl get pvc -n {{ .Release.Namespace }}

To check logs for a specific database:

  kubectl logs -n {{ .Release.Namespace }} postgres-user-0
  kubectl logs -n {{ .Release.Namespace }} postgres-content-0
  kubectl logs -n {{ .Release.Namespace }} postgres-media-0
  kubectl logs -n {{ .Release.Namespace }} postgres-category-0
  kubectl logs -n {{ .Release.Namespace }} postgres-comment-0

To connect to a database:

  kubectl exec -it -n {{ .Release.Namespace }} postgres-user-0 -- psql -U user_service -d user_service
